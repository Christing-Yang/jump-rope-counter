## How to Use

1. Open the [demo](https://christina-yang.github.io/jump-rope-counter/) in Safari on your iPhone
2. Click "Start Detection" and allow camera access
3. Position your phone to capture your jumping motion
4. Start jumping! The counter will automatically track your jumps
5. Use the controls to pause, reset, or switch cameras

## Technical Details

- Built with HTML5, CSS3, and vanilla JavaScript
- Uses `getUserMedia` API for camera access
- Implements motion detection algorithm based on frame differences
- Optimized for mobile Safari performance
- Responsive design for all iPhone screen sizes

## Privacy Notice

This application runs entirely in your browser. No data is sent to any servers. Your camera feed is processed locally and never leaves your device.
